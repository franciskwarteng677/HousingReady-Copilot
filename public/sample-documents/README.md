# Synthetic sample documents

This folder contains the three fictional PDF fixtures used by the Profile
workflow:

- synthetic-pay-stub.pdf
- synthetic-benefits-letter.pdf
- synthetic-residency-document.pdf

Every PDF is visibly marked:

SYNTHETIC DEMO DOCUMENT — NOT REAL APPLICANT DATA

The deterministic demo extractor recognizes these exact files by SHA-256
fingerprint. Do not add real applicant records, identity documents, financial
information, or other personal data.
